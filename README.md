# Traversal Projesi

Traversal, ASP.NET Core MVC kullanılarak geliştirilmiş, **dinamik bir tur ve seyahat yönetim sistemi**dir.  
Proje; ziyaretçi, üye ve admin panelleri ile gerçek hayata uygun, kurumsal mimaride tasarlanmıştır.

---

## 🚀 Proje Özellikleri

### 🌍 Ziyaretçi (Site) Tarafı
- Dinamik **Anasayfa**
- **Tur Rotaları** listeleme
- **Tur Rota Detayları**
- Her tura özel:
  - Tur rehberi bilgileri
  - Yorumlar
  - Yorum yapma işlemi
- **İletişim Sayfası**

---

### 👤 Üye Paneli
- 🌐 **Çoklu Dil Desteği (Localization)**
- 👤 Profil görüntüleme ve güncelleme
- 📌 **Rezervasyon Yönetimi**
  - Aktif rezervasyonlar
  - Geçmiş rezervasyonlar
  - Onaylanan rezervasyonlar
- 🧾 İlgili tur için **rezervasyon oluşturma**
- 🗺️ Tur rotalarını görüntüleme
- 💬 Yapılan yorumları görüntüleme

---

### 🛠️ Admin Paneli
- 📊 **Dashboard** (istatistikler)
- 🗺️ Tur Rotaları 
- 💬 Yorumlar 
- 🧑‍🤝‍🧑 Misafir / Kullanıcı yönetimi
- 🧭 Rehber yönetimi
- 📩 İletişim sayfasından gelen mesajlar
- 📧 Site üzerinden **mail gönderme**
- 📑 **Excel & PDF raporlama**
- 🔐 **Rol yönetimi**
- Tüm modüller **CRUD** mantığıyla geliştirilmiştir

---

## 🔐 Kimlik Doğrulama & Güvenlik
- ASP.NET Core Identity
- Authentication & Authorization
- Rol bazlı yetkilendirme
- **Forgot Password / Reset Password** işlemleri (mail ile)

---

## 🧩 Mimari & Tasarım Desenleri
- N-Tier Architecture
- Repository Design Pattern
- CQRS Design Pattern
- MediatR
- DTO (Data Transfer Object)
- AutoMapper
- Dependency Injection
- Fluent Validation
- SignalR (gerçek zamanlı işlemler)

---

## 🔗 API Entegrasyonu
- RapidAPI kullanılarak **Booking / Otel verileri** çekilmiştir
- API tüketimi servis katmanında gerçekleştirilmiştir

## 📌 Projenin Amacı
Traversal projesi; **ölçeklenebilir, sürdürülebilir ve kurumsal mimariye uygun** bir tur yönetim sistemi geliştirmeyi amaçlamaktadır.  
Gerçek dünya senaryolarına uygun olarak admin, üye ve ziyaretçi panelleri bir arada tasarlanmıştır.
